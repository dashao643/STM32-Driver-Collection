# STM32 轻量级外设驱动合集

一些自己开发的关于STM32F1系列的入门级外设驱动库。

## 目前已包含：
1. 128*64像素OLED
2. 非阻塞定时器按键
3. DHT11驱动
4. PWM常用案例
5. Modbus简易协议通信（支持04、05、10功能码）
6. RTC外设实时时钟
7. 数码管驱动
8. WiFi（esp8266，AP模式）
9. RS485（待测试）
10. ADC采集
11. 软件模拟I2C

## 未来：

15. 以太网通信协议应用

17. 蓝牙、4G、2.4G、LoRa、NB、Zigbee
18. IAP、OTA升级
19. SD卡、IC卡
20. 超声波
