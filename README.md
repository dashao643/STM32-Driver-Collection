# STM32 轻量级外设驱动合集

一些自己开发的关于STM32F1系列的入门级外设驱动库。

## 目前已包含：
1. 128*64像素OLED
2. 非阻塞定时器按键
3. DHT11驱动
4. PWM常用案例
5. Modbus简易协议通信（支持04、05、10功能码）
6. RTC外设实时时钟
7. 数码管驱动
8. WiFi（esp8266，AP模式）

## 未来：
9. IAP、OTA升级
10. DS18B20驱动
11. PID电机控制
12. RS485、CAN通信
13. MQTT、HTTP、CoAP、DDS
14. 红外通信、RFID、NFC
15. 以太网通信协议应用
16. LCD屏幕驱动（8080协议）
17. 蓝牙、4G、2.4G、LoRa、NB、Zigbee
