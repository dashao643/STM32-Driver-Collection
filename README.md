# STM32 轻量级外设驱动合集

一些自己开发的关于STM32F1系列的入门级外设驱动库。

## 目前已包含：
1. 128*64像素OLED
2. 非阻塞定时器按键
3. DHT11驱动
4. PWM常用案例
5. Modbus简易协议通信（支持04、05功能码）

## 未来：
6. LCD屏幕驱动（8080协议）
7. 蓝牙、WiFi、4G、2.4G、LoRa、NB、Zigbee
8. IAP、OTA升级
9. 数码管驱动
10. DS18B20驱动
11. PID电机控制
12. CAN通信测试
13. MQTT、HTTP、CoAP、DDS
14. 红外通信、RFID、NFC
15. 以太网通信协议应用
